# Superstore Sales Analysis

## Project Overview

This project analyzes the Sample Superstore dataset using SQL. The goal is to explore sales performance, profit trends, and product category insights to better understand business performance.

## Tools Used

* PostgreSQL
* SQL
* Power BI

## Dataset

The dataset used in this project is the **Sample Superstore dataset**, which contains information about orders, products, sales, profit, and customer segments.

## SQL Analysis

The SQL queries in this project include:

* Data cleaning and type conversion
* Total sales and profit calculation
* Sales by category and sub-category
* Monthly sales trend analysis
* Profitability analysis by region and product

## Dashboard

A Power BI dashboard was created to visualize key metrics such as total sales, profit, and sales distribution across categories.

## Key Insights

* Technology category generates the highest sales.
* Some sub-categories have high discounts but low profit.
* Sales trends vary across different regions.
